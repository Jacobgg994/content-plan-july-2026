## 🥇 ทำไม LINE OA ถึงสำคัญที่สุด

```
LINE users ไทย:     40M+ (อันดับ 2 ของโลก)
เปิดอ่าน (open rate): 70-80% (email แค่ 5-10%)
พฤติกรรมไทย:         ซื้อของผ่าน LINE > email, แชทก่อนซื้อ
```

> LINE OA = CRM + Broadcast + Chatbot + ขายของ — ในแอปเดียว
> ถ้าทำ LINE OA ดี = ต้นทุนต่ำ แต่ conversion สูงสุด

## 🕐 กฎการส่ง

| กฎ | เพราะ |
|-----|--------|
| ✅ ส่งสูงสุด 3 ครั้ง/สัปดาห์ | มากกว่านี้ = unblock |
| ✅ ส่ง อังคาร + พฤหัส + เสาร์ (ถ้ามีโปร) | จันทร์คนยุ่ง, ศุกร์คนเลิกงาน |
| ✅ เวลา 12:00 หรือ 20:00 | พักเที่ยง / หลังเลิกงาน |
| ✅ รูปก่อนข้อความ เสมอ | Engagement สูงกว่าข้อความอย่างเดียว |
| ✅ ข้อความ 2-3 บรรทัดเท่านั้น | คนไทยอ่าน LINE แบบเร็ว |
| ✅ มีปุ่ม Rich Message ทุกครั้ง | ลดขั้นตอน = CTR สูงกว่า |
| ❌ ห้าม text-only (ไม่มีรูป) | คนไม่เปิด |
| ❌ ห้ามเกิน 4 บรรทัด | คนไม่ scroll |
| ❌ ห้ามภาษาแข็ง/ราชการ | ดูไม่น่าเชื่อถือ |
| ❌ ห้าม >3 ครั้ง/สัปดาห์ | โดน block ถาวร |

## ⚠️ ข้อควรระวัง

| สิ่งที่ห้าม | เพราะ |
|------------|--------|
| ❌ ส่งเกิน 3 ครั้ง/สัปดาห์ | LINE OA จำกัด — เกิน = โดนบล็อก |
| ❌ ข้อความยาวเกิน 4 บรรทัด | คนไม่อ่าน — scroll ผ่าน |
| ❌ ไม่มีรูป | Open rate ต่ำ — คนไม่สนใจ |
| ❌ ภาษาแข็ง / ภาษาราชการ | ไม่น่าเชื่อถือในบริบท LINE |
| ❌ สัญญาเกินจริง | PDPA + consumer protection |
| ❌ ขายอย่างเดียวทุกข้อความ | คนเบื่อ → unblock |
| ❌ ส่งตอนตี 1-ตี 5 | รบกวน → unblock + เสียภาพลักษณ์ |

## 🔒 PDPA Compliance

```
✅ ทุก broadcast ต้องมี opt-out (unfollow) ชัดเจน
✅ เก็บเฉพาะข้อมูลที่จำเป็น (ชื่อ, LINE ID, การสมัคร)
✅ ลบข้อมูลเมื่อลูกค้าขอ — ภายใน 30 วัน
✅ ไม่ส่งต่อข้อมูลให้ third party
✅ บันทึก consent ตอนลูกค้า add friend ครั้งแรก
```
