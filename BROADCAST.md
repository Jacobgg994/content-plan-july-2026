## 🔁 Broadcast Workflow

```
วันก่อนส่ง (จันทร์ / พุธ / ศุกร์):
  SALMON เขียนข้อความ + เตรียมรูป → KOMHAS ตรวจ → ตั้งเวลาใน LINE OA

วันส่ง (อังคาร / พฤหัส / เสาร์):
  12:00 — ส่ง (หรือ 20:00 ถ้าเป็น working hour audience)

หลังส่ง 24 ชม.:
  KOMHAS เช็ค open rate + CTR → บันทึกใน weekly review
```

## ⚠️ ข้อควรระวัง

| สิ่งที่ห้าม | เพราะ |
|------------|--------|
| ❌ ส่งเกิน 3 ครั้ง/สัปดาห์ | LINE OA จำกัด — เกิน = โดนบล็อก |
| ❌ ข้อความยาวเกิน 4 บรรทัด | คนไม่อ่าน — scroll ผ่าน |
| ❌ ไม่มีรูป | Open rate ต่ำ — คนไม่สนใจ |
| ❌ ภาษาแข็ง / ภาษาราชการ | ไม่น่าเชื่อถือในบริบท LINE |
| ❌ สัญญาเกินจริง | PDPA + consumer protection |
| ❌ ขายอย่างเดียวทุกข้อความ | คนเบื่อ → unblock |
| ❌ ส่งตอนตี 1-ตี 5 | รบกวน → unblock + เสียภาพลักษณ์ |

## 🔒 PDPA Compliance

```
✅ ทุก broadcast ต้องมี opt-out (unfollow) ชัดเจน
✅ เก็บเฉพาะข้อมูลที่จำเป็น (ชื่อ, LINE ID, การสมัคร)
✅ ลบข้อมูลเมื่อลูกค้าขอ — ภายใน 30 วัน
✅ ไม่ส่งต่อข้อมูลให้ third party
✅ บันทึก consent ตอนลูกค้า add friend ครั้งแรก
```
